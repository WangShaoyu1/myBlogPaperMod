简介
--

`Arco-admin-template`是一个免费开源的中后台模板你，使用了`React17`、`vite2`、`react-router-dom v6`、`less` 等主流技术开发，开箱即用的中后台前端解决方案。

项目目的：

*   学习`React Hook`相关 Api
*   掌握`Vite2`插件机制、构建配置
*   学习 `Ract-router-dom v6`新特性
