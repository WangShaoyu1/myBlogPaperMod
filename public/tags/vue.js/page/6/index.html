Electron的优势 开源的核心扩展比较容易，加之现在 gyp 已经非常人性化了，使得c++ 和 js 搞基非常容易。 界面定制性强，原则上只要是Web能做的他都能做。 是目前最廉价的跨平台技术方案，HTML+JS 有大量的前端技术人员储备，而且有海量的现存web UI 库。大多都很靠谱。 相对其他跨平台方案（如 QT GTK+ 等），更稳定，bug少， 毕竟只要浏览器外壳跑起来了，里面的问题不会太多 ，当然我也遇到过一些暗坑。 方便热更新。下载覆盖完事。当然这是所有脚本共有的优点。 4.
