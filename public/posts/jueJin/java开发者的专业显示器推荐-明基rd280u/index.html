哈喽，小伙伴们好呀，我混编程界已经好几年了，搞了这么多年的Java，换了好几家公司，有的公司发电脑，有的公司发笔记本，有的还发主机，不过你们有没有注意到一个普遍的问题，公司发的显示器一般都不怎么样，我遇到的大部分都是1080p的，很便宜的那种，就像这种

这种显示器剩下的是老板的钱，但是废的是吗喽的寿命，上面说了，公司为了省钱发的都是很便宜的显示器，对咱们的眼睛不好，看久了眼睛酸痛肿胀。然后导致工作都难受。再来说说第二个问题，市面上大部分都是16:9的显示器，搞编程的都知道，特别是像我这样的Java开发者们更清楚，打开IDEA，启动服务，下面的控制台占了三分之一，左边的项目视图占了四分之一，留给咱们写代码的地方就没多少了！！第三个，它不防蓝光，不防蓝光，不防蓝光！！！眼睛看着都疼，还有如果坐在窗户边或者灯光下的也知道，那光反射导致的一片区域看不清真的难受。

市面上有专门为游戏设计的、为电影设计的、为画图设计的都有，但是就是没有专门针对编程而出现的显示器，而且上面的痛点是作者写代码过程非常不舒服的点，但是现在终于有了世界上第一款专门为编程设计的显示器，那就是明基RD280U显示器！！！

在知道我们有这样的困扰之后，感谢金主爸爸寄过来的一台显示器

金主爸爸说他们是全球首款为编码人士打造的专业编程显示器，并且RD280U在设计中使用了至少85%的消费后回收塑料制成。真的respect！！

用了一段时间后，我来分享一下我的使用感受：真的很不错。既然说它是专门为编程设计的专业显示器，那它到底专业在哪呢？？

我们来看看普通的显示器长啥样，就是我笔记本电脑自带的显示器，这个甚至从上往下看都是白屏！！

而这款显示器360度无死角，超级清晰：

现在我们来开启明基RD280U显示器专业模式：

整体可能看不出来细致的差别，我们再来看一下细节：

代码清晰度：4K画质+抗反射面板#

刚才作者说得难受的地方就是这个了，1080P加灯光反射，这俩东西一起搞，想看清代码都费劲。但是，明基RD280U不怕这个，而且它还是4k的，一下子就从png图片变成svg图片了，而且它的面板有抗反射涂层，坐在窗户边和灯光下都不怕了，代码依旧显示的清楚。（我们公司窗外风景挺好的，但大家都把窗帘拉上，就是因为看不清代码，浪费了这么好的风景，建议我们老板跟明基进货）

3:2方屏代码显示区域更大#

现在市面上的显示器大部分都是16:9的比例加上14寸的小屏幕，就像作者前面说的：打开IDEA，启动服务，下面的控制台占了三分之一，左边的项目视图占了四分之一，留给我们写代码的地方就不多了！！如果右边的工具再打开，那就没法看了，就像这样：

图片不来自网络

咱吹牛没用，直接看效果，3:2的生产力方屏，28寸的大屏，不只是数字吓人，实际用起来更吓人。再给大家看个动图，注意看编辑区的占比，一下子放大了很多，编辑的代码行数实际增加了不少，也减少了查看代码的滚动次数。

更适合加班人的体质：Moonhalo智慧光环和夜间防护#

这款显示器有个背光板，没错，自带的！他们叫它MoonHalo智慧光环，这个背光设计真的很贴心，主打就是给晚上还要写代码的程序员提供沉浸式的工作环境里。

而且减少了夜晚环境光和屏幕光的对比眩光，让深夜开发变得更加舒适和专注

除此之外，还有一个夜间保护功能，也叫猫头鹰模式，不管周围环境多暗，RD280U都能通过固件调整打破亮度限制，提供最低的亮度和色彩平衡方案。在极低亮度下代码依旧非常清晰

而且在护眼方面，RD280U是有专属的莱茵认证：Eyesfae2.0、 Eyecomfort、无频闪、硬件级滤蓝光和抗反射认证。在护眼方面，是专业的！

专门为编程模式设计的实体按键#

就在logo的正下方，里面有个编码快捷触键，能一键调出显示屏的各种模式，比如前面提到的编程模式、猫头鹰模式，模式切换起来非常方便。悄悄告诉你，这块区域还有传感器，人走了显示器就会自动息屏！！！保护我们的代码隐私

专门的软件#

除了有实体按键之外，还可以搭配BenQ研发的Display Pilot 2软件一键调试你想要的屏幕模式，真的很人性化。这个软件甚至精细到了可以给每一款软件设置不同的颜色模式，嗯，有点东西。。

保护肩颈的人体工学设计#

这个功能真的很实用，如果你的老板没给你配升降桌（至少我没见过配升降桌的老板），这款显示器可以高低升降，我试过了站起来之后升到最高处完全没问题，而且很有阻尼感，非常好，前后倾斜，保护长时间看屏幕的人的肩颈健康。

而且RD280U还配有人体工学支架款RD280UA，如果想追求更多角度的就可以选择官方支架款。

以前用的都是公司配的显示器或者直接只有一个笔记本电脑，看久了眼睛疼得不行，导致经常要买眼药水滴，但是这款显示器因为有抗蓝光加上显示清晰，所以用的这半个月感觉特别舒服。身边也有很多同事问我这个显示器怎么这么大呢，还有这个比例怎么和我们的显示器不一样啊之类的。有需要的小伙伴可以试试
